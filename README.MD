# 📌 Dynamic Home Page with Hero Slider

## 🎯 Project Overview
This project is a modern, responsive home page featuring a dynamic hero slider. The slider highlights different projects or services with images, titles, descriptions, and call-to-action buttons. Slides are generated dynamically from a JSON file.

The goal of this project is to demonstrate front-end development skills, including layout design, interactivity, animations, and dynamic content rendering using JavaScript.

## 🛠 Technologies Used
* **HTML5:** Used for structuring the web page using semantic elements like `<section>`, `<h1>`, and `<p>`.
* **CSS3:** Used for styling, layout, responsiveness, animations, and hover effects.
* **JavaScript (Vanilla JS):** Used to fetch data from JSON file, dynamically generate slides, and control slider functionality (auto-slide, navigation, dots).
* **JSON:** Used to store slide data such as title, description, image, and button text.

## ⚙️ Features
* Dynamic hero slider (data loaded from JSON)
* Cinematic auto-slide rotation (every 5 seconds)
* Manual navigation (previous/next arrows)
* Minimalist slide indicators (dots)
* Premium Ken Burns zoom and staggered fade transitions
* Glassmorphism button hover effects
* Fully responsive design (desktop, tablet, mobile)

## ▶️ How to Run
1. Open the project folder in VS Code.
2. Install the **Live Server** extension.
3. Right-click on `index.html`.
4. Click **Open with Live Server**.

## ✅ Conclusion
This project demonstrates the implementation of a responsive and interactive hero slider using modern front-end technologies along with dynamic data handling.
