Dynamic Home Page with Hero Slider
📌 Project Overview

This project is a modern, responsive home page featuring a dynamic hero slider. The slider highlights different projects or services with images, titles, descriptions, and call-to-action buttons. Slides are generated dynamically from a JSON file.

🎯 Objective

The goal of this project is to demonstrate front-end development skills, including layout design, interactivity, animations, and dynamic content rendering using JavaScript.

🛠 Technologies Used
HTML5
Used for structuring the web page using semantic elements like <section>, <h1>, and <p>.
CSS3
Used for styling, layout, responsiveness, animations, and hover effects.
JavaScript (Vanilla JS)
Used to:
Fetch data from JSON file
Dynamically generate slides
Control slider functionality (auto-slide, navigation, dots)
JSON
Used to store slide data such as title, description, image, and button text.
⚙️ Features
Dynamic hero slider (data loaded from JSON)
Auto slide rotation (every 4 seconds)
Manual navigation (previous/next arrows)
Slide indicators (dots)
Smooth fade transition between slides
Button hover effects (color change and scale)
Fully responsive design (desktop, tablet, mobile)

project/

▶️ How to Run
Open the project folder in VS Code
Install the Live Server extension
Right-click on index.html
Click Open with Live Server
📌 Notes
The slider content is fully dynamic and can be updated by editing the slides.json file.
Images used in the project are placeholders and can be replaced with real project images.
✅ Conclusion

This project demonstrates the implementation of a responsive and interactive hero slider using modern front-end technologies along with dynamic data handling.
